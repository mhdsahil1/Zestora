# Zestora – Full Stack E-Commerce Platform

Zestora is a full-stack e-commerce web application built using Next.js, MongoDB, and NextAuth.  
It includes authentication, admin dashboard, product management, cart, checkout, Razorpay payment integration, order management, and inventory tracking.

## Tech Stack
- Next.js (App Router)
- MongoDB + Mongoose
- NextAuth (Google + Credentials)
- Razorpay Payment Gateway
- Tailwind CSS
- Zod Validation

## Features
- User Authentication (Google OAuth + Email/Password)
- Role-Based Admin Dashboard
- Product Catalog with Search & Filters
- Cart and Wishlist
- Checkout with Address & Payment Options
- Razorpay Payment Integration
- Order Management & Order Tracking
- Inventory Management
- Order Cancellation with Inventory Rollback
- User Profile Management
- Admin Order Status Management

## System Architecture
Frontend → Next.js  
Backend → Next.js API Routes  
Database → MongoDB  
Auth → NextAuth  
Payment → Razorpay
